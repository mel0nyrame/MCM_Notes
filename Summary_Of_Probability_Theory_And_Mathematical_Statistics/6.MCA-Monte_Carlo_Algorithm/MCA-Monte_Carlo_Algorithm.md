# 蒙特卡洛算法

## 算法概述

核心: 用随机性解决确定性的思想

定义: 用大量随机试验来模拟中子扩散问题特定过程

## 概率论基础

随机事件: 结果不确定的现象

概率: 事件发生可能性的度量

基本性质: $ 0 \leq P(A) \leq 1$

随机变量:

- 离散型随机变量 $\quad x \in {0,1}$
- 连续性随机变量 $\quad x \in [0,1]中的任意值$

常见分布:

- 均匀分布: 等概率出现
- 正态分布: 钟型对称分布
- 指数分布: 等待时间分布

## 重要定理

### 1.大数定律(保证了蒙特卡洛算法的收敛性)

当试验次数足够多,事件出现的频率会趋近于理论概率

\[
\lim_{n \to \infty} \frac{x_1+x_2+\dots+x_n}{n} = \mu
\]

### 2.中心极限定律

无论原始分布如何,样本均值分布总是趋于正态分布

\[
X~N(\mu,\frac{\sigma^2}{n}) \quad n \to \infty
\]

## 核心思想(关注估计值,计算概率/期望,求平均)

### 基本理念:随机模拟+统计推断

### 关键要素

- 随机生成
- 数学建模
- 统计估计

### 工作原理

- 问题抽象化: 问题转换成概率或期望计算
- 模型建立: 设计随机试验
- 随机抽样: 生成随机样本模拟
- 统计分析: 计算样本统计量为目标估计

### 场景

- 概率/期望计算(计算$\pi$,积分,计算复杂几何体)
- 复杂系统模拟(金融市场,物理粒子运动)
- 高维优化(神经网络优化)
- 难以精确建模(工程可靠性)

### 检验清单

- 问题目标可以表示为期望值？
- 使用了随机数生成器？
- 进行了大量独立试验？
- 通过样本均值估计目标值？
- 估计精度随样本量增加而提高？
- 能计算置信区间或估计误差？

## 算法框架

1. **构造概率模型**
   - 将目标问题转化为概率或期望值计算问题
   - 目标量 = \(E[g(X)]\)

2. **随机抽样**
   - 根据概率模型生成大量随机样本
   - \(X_1, X_2, \ldots, X_n \sim \) 某种分布

3. **统计计算**
   - 对样本进行统计分析，计算相关量
   - 计算 \(g(X_1), g(X_2), \ldots, g(X_n)\)

4. **结果估计**
   - 用样本统计量作为目标量的估计
   - 估计值 = \(\left(\frac{1}{n}\right) \sum g(X_i)\)

## 模型建立步骤

### 第一步：确定问题

- 什么类型的问题？
- 求解的目标量是什么？

### 第二步：构建概率模型

- 定义随机变量及其分布
- 建立目标量与随机变量的关系
- 确保模型的数学正确性

### 第三步：设计随机试验

- 确定抽样方案
- 选择合适的随机数分布
- 考虑是否需要方差缩减技术

### 第四步：模拟循环和统计计算

1. 随机数生成
2. 计算估计值
3. 估计误差和置信区间
4. 收敛性检验
